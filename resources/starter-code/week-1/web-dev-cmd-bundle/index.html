Wildcards `*` exercise

Challenge: Move the images to the correct folder

Explanation: The image files are all in the root folder of this project. You must create the directory structure with `mkdir`. Then move the images to the correct folder with `mv` command.

Dice

The following images are all PNG files. Try creating a new folder called `img` and a subfolder called `dice`. Then move the images to the correct folder with `mv` command.

You can move a single file like this: 

        mv dice-1.png img/dice/dice-1.png

Or you can move all the files in a folder with a wildcard `*`like this:

        mv img/dice/*.png img/dice/

Explanation: The wildcard `*` is a special character that matches all characters. We use it to match a pattern within filenames.

Birthday

The following images are all JPG files. Try creating a new folder called `img` and a subfolder called `memes`. Then move the images to the correct folder with `mv` command. Try using a wildcard.

Landscapes

In the previous examples, you could use the wildcard to replace the filename, while keeping the file extension (`*.png` or `*.gif`). This won't work here because we want to divide the JPEG files into multiple folders. Try to think of a way to move all the beach files to a new folder with a wildcard.
